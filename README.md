# 💰 Análise de Dados Bancários com Python (Excel & SQL)

Este repositório contém soluções para desafios de programação realizados durante o **Bootcamp GenAI & Dados**, promovido pela DIO em parceria com o Bradesco.

Os desafios simulam situações reais do mercado financeiro, envolvendo análise de dados, uso de Excel e consultas SQL no contexto bancário.

---

## 🏦 Contexto

Durante o bootcamp, foram propostos desafios práticos voltados para o dia a dia de analistas de dados em instituições financeiras, abordando:

- Análise de crédito  
- Monitoramento de transações  
- Simulação de empréstimos  
- Controle de gastos de clientes  

---

## 📌 Desafios

### 🔹 Parte 1: Operações de Dados para Análise Bancária

Simula o uso de Excel e SQL para análise de dados bancários, como:

- Relatórios de juros  
- Limite de crédito de clientes  
- Consulta de saldo  
- Histórico de transações  

📂 Arquivo: `parte1_operacoes_bancarias.py`

---

### 🔹 Parte 2: Simulações Bancárias com Excel e SQL

Simula cenários analíticos e financeiros, como:

- Simulação de parcelas de empréstimos  
- Resumo de gastos de clientes  
- Total de compras no cartão  
- Empréstimos em aberto  

📂 Arquivo: `parte2_simulacoes_bancarias.py`

---

## 🧠 Conceitos aplicados

- Estruturas condicionais (`if/elif/else`)  
- Funções em Python  
- Entrada e saída de dados (STDIN/STDOUT)  
- Lógica aplicada ao contexto financeiro  
- Simulação de consultas SQL  
- Uso conceitual de Excel para análise
